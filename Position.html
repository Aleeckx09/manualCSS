La propiedad position especifica el tipo de metodo de posicionamiento utilizado para un elemento(estático, relativo, fijo o absoluto)


Propiedad position

  • static: Es el valor por defecto. Los elementos se posicionan según el flujo normal del documento.
  • relative: El elemento se posiciona en relación con su posición normal en el flujo del documento.
  • fixed: El elemento se posiciona en relación con la ventana del navegador, permaneciendo fijo incluso al desplazar la página.
  • absolute: El elemento se posiciona en relación con el primer ancestro posicionado (no estático), o con el bloque contenedor si no hay ningún ancestro posicionado.
  • sticky: El elemento se posiciona en relación con el área de desplazamiento del contenedor más cercano y se comporta como relative hasta que alcanza un límite específico, momento en el que se vuelve fijo.
